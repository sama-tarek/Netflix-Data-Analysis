# Netflix Data Analysis Project

## Overview
Exploratory data analysis of Netflix titles dataset using Python.

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate catalog
- USA leads content production
- Rapid growth after 2015

## Final Conclusion
1. Netflix focuses mainly on movies.
2. USA is the largest content producer.
3. Platform growth accelerated after 2015.
4. Mature audience ratings dominate.
5. Most movies are medium length.
