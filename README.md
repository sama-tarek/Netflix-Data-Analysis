# Netflix Data Analysis

## Overview

This project explores the Netflix dataset to uncover trends and insights about movies and TV shows available on the platform. Using Python and data visualization techniques, the analysis examines content distribution, ratings, genres, release years, and other characteristics of Netflix's catalog.

## Objectives

* Perform exploratory data analysis (EDA).
* Clean and preprocess the dataset.
* Identify trends in Netflix content.
* Visualize important patterns and relationships.
* Extract meaningful insights from the data.

## Dataset Features

The dataset contains information about Netflix titles, including:

* Type (Movie or TV Show)
* Title
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Genre
* Date Added

## Data Cleaning

The following preprocessing steps were performed:

* Handled missing values.
* Removed duplicate records.
* Converted columns to appropriate data types.
* Prepared the dataset for analysis and visualization.

## Exploratory Data Analysis

The project investigates:

* Distribution of Movies vs TV Shows
* Content ratings
* Release year trends
* Top countries producing Netflix content
* Most common genres
* Duration analysis
* Content added over time

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Insights

* Movies make up the majority of Netflix content.
* The number of titles added to Netflix has increased significantly over recent years.
* Certain countries contribute the largest share of content.
* Some genres are considerably more popular than others.
* Content ratings vary depending on the target audience.
